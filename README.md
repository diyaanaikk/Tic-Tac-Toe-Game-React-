# Tic Tac Toe Game (React)

A simple and interactive Tic Tac Toe game built using React.  
This project demonstrates the use of React Hooks, state management, and basic game logic.

## Features

- Two-player gameplay (X vs O)
- Real-time winner detection
- Draw detection
- Reset game option
- Clean and responsive UI
- 
## Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
